# DigiCV — The Architect

Live portfolio and digital CV for an operational systems specialist with 10+ years across fraud prevention, telecoms, insurance and energy.

**Live site:** https://the-architect-neo.github.io/portfolio/

---

## What it is

A single-page portfolio built in HTML/CSS. No frameworks. No dependencies. Deploys direct to GitHub Pages.

Sections:
- Professional identity and headline
- Career experience with skill tags
- Key metrics (years, industries, tools)
- Active ventures and projects with live/building/held status indicators
- Contact and social links

## Stack

- Pure HTML5 + CSS custom properties
- Space Grotesk + Space Mono (Google Fonts)
- Light/dark theme toggle via `data-theme`
- Schema.org structured data for SEO
- GitHub Pages hosting

## Deploy

1. Push to `main`
2. GitHub Pages serves from root
3. No build step required
